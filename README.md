# SP 36.13330 – Pipeline Parameters Calculator

Инструмент для расчета основных параметров магистрального газопровода по СП 36.13330 «Магистральные трубопроводы» с возможностью формирования структурированного инженерного отчета для включения в проектную документацию.

---

## Практический контекст

В рамках разработки проектной документации по магистральному газопроводу необходимо было:

- определить основные расчетные параметры газопровода по СП 36.13330 (гл. 12);
- обосновать выбранные параметры перед руководством и заказчиком;
- подготовить аккуратно оформленный расчетный отчет, который можно напрямую включить в ПД/РД.

Для сокращения времени на ручные расчеты, минимизации ошибок и унификации отчетов была разработана данная программа которая:

- принимает исходные данные по трубопроводу, материалу и условиям работы;
- выполняет полный цикл расчётов по главе 12 СП 36.13330.2012;
- формирует структурированный отчёт с отсылками к пунктам норм, который можно напрямую использовать в пояснительной записке
  
Программа позволяет быстро получать результаты по нескольким вариантам параметров и формировать отчет, понятный как инженерам, так и руководству.

---

## Функционал

Приложение реализует основные расчёты главы 12 СП 36.13330.2012 для магистральных трубопроводов:

- Расчёт толщины стенки (п. 12.3)
- Проверка прочности и устойчивости (п. 12.4)
- Устойчивость против всплытия (п. 12.4.5)
- Расчет компенсаторов (п. 12.6)
- Расчёт для сейсмических районов (п. 12.7)
